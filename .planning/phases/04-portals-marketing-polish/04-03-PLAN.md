<task type="auto">
  <name>Task 2: JSON-LD schema markup across site</name>
  <files>src/components/seo/JsonLd.tsx, src/lib/schema-markup.ts, src/app/(shop)/products/[slug]/page.tsx, src/app/(blog)/blog/[slug]/page.tsx, src/app/layout.tsx</files>
  <action>
1. Create `src/components/seo/JsonLd.tsx`:
   - Generic component: `JsonLd<T extends Thing>({ data }: { data: WithContext<T> })`.
   - Renders `<script type="application/ld+json">` with JSON.stringify(data).
   - Type-safe using schema-dts types.

2. Create `src/lib/schema-markup.ts` with helper functions:
   - `productSchema(product)`: Returns Product JSON-LD with name, sku, description, brand (Torke), manufacturer, offers (price if available, InStock availability), image.
   - `articleSchema(post)`: Returns Article JSON-LD with headline, description, datePublished, author (Organization: Torke), publisher, image.
   - `organizationSchema()`: Returns Organization JSON-LD for Torke (name, url, logo, contactPoint).
   - `breadcrumbSchema(items: {name, url}[])`: Returns BreadcrumbList JSON-LD.
   - `faqSchema(items: {question, answer}[])`: Returns FAQPage JSON-LD (for glossary, Plan 04).
   - `howToSchema(steps)`: Returns HowTo JSON-LD (for installation guide articles).

3. Add Product JSON-LD to `src/app/(shop)/products/[slug]/page.tsx`:
   - Import JsonLd and productSchema.
   - Render `<JsonLd data={productSchema(product)} />` in the page.
   - Add BreadcrumbList: Home > Products > [Category] > [Product Name].

4. Add Article JSON-LD to `src/app/(blog)/blog/[slug]/page.tsx`:
   - Render `<JsonLd data={articleSchema(post)} />`.
   - Add BreadcrumbList: Home > Blog > [Post Title].

5. Add Organization JSON-LD to root `src/app/layout.tsx`:
   - Render `<JsonLd data={organizationSchema()} />` in the root layout so it appears on every page.
