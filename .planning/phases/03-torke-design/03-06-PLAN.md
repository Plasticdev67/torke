    1. Create ProductRecommendations.tsx:
       - "use client" component
       - Reads anchorType, anchorDiameter, groupPattern from useDesignStore
       - Uses tRPC to query products matching the anchor type and diameter:
         - For chemical anchors: query products in "chemical-anchors" category matching diameter
         - For mechanical anchors: query products in "mechanical-anchors" category matching diameter
         - Use existing products.list or products.listByCategory tRPC query with appropriate filters
       - Displays 1-3 product cards:
         - Dark card (#1A1A1A) with product image, name, SKU, price (formatted as GBP)
         - "Add to Cart" button (red #C41E3A)
         - Quantity input pre-filled from group size: single=1, 2x1=2, 2x2=4, 3x2=6
         - User can adjust quantity before adding
       - "Add to Cart" onClick: calls useCartStore.addItem with:
         - productId, productName, sku, quantity, unitPricePence from the product
         - calcReference from useDesignStore.calcReference (if saved) or a temporary reference
       - Show "Recommended Products" header with brief explanation: "Based on your M16 chemical anchor design"
       - If no matching products found: show "No matching products found. Browse catalogue" link to /products
       - Section sits below the ResultsPanel in the right panel
       - Uses same product database as e-commerce (DESIGN-19) -- no separate product list

    2. Update orders router (src/server/trpc/routers/orders.ts):
       - In the createOrder mutation, when processing cart items, pass through calcReference to the orderLines insert
       - Add calcReference field to the orderLine insert: `calcReference: item.calcReference ?? null`
       - This wires DESIGN-16: calculation reference linked to resulting order
